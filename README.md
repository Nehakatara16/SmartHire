#  SmartHire – AI Resume Screening System

SmartHire is a web-based application that automates resume screening using AI. It helps recruiters efficiently analyze, rank, and manage candidates based on their skills and scores.

## Features

*  Resume Upload & Parsing
*  AI-Based Candidate Scoring
*  Dashboard with Analytics & Charts
*  Search & Filter Candidates
*  Ranking System
*  Admin Panel for Status Updates
*  Smart Notifications (Selected, Rejected, Pending)

##  Tech Stack

* **Frontend:** HTML, CSS (Glassmorphism UI), Bootstrap
* **Backend:** Flask (Python)
* **Database:** MySQL
* **Libraries:** Chart.js, SweetAlert, Toastify, Confetti

## How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Run the app:

   ```
   python app.py
   ```
4. Open in browser:

   ```
   http://127.0.0.1:5000
   ```

## Use Case

Helps recruiters save time by automatically screening resumes and identifying the best candidates.
